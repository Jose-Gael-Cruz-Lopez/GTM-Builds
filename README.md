# GTM-Builds

El Sistema es una plataforma de dos agentes conectados que cierra el loop entre marketing y ventas usando RAG — Retrieval Augmented Generation — como su núcleo técnico. El primer agente, El Publicador, toma la URL de un blog o sitio web, recupera en tiempo real el contenido del artículo y la identidad visual de la marca (colores, logo, tono), y usa ese contexto para generar un carrousel listo para publicar en LinkedIn o Instagram sin que el marketer configure nada. El segundo agente, El Comité, toma el dominio del prospecto, recupera su propuesta de valor, noticias recientes y contexto del sitio, y además consulta una vector database de objeciones históricas — deals perdidos anteriores — para generar un champion brief hiperpersonalizado, tarjetas de objeciones con rebuttals basados en datos reales, y una nota de voz de 60 segundos lista para compartir. El loop de retroalimentación es donde el RAG se vuelve inteligente: cuando un deal se pierde por una objeción, esa objeción se embeds en el vector store; la próxima vez que El Publicador genera contenido, recupera esas objeciones y las inyecta en el prompt para que el LLM genere un carrousel que las contrarreste directamente en LinkedIn antes de que el próximo comprador las tenga. El resultado es un sistema GTM que aprende solo de cada deal ganado y perdido: una URL entra, el contexto correcto se recupera, y el output sale personalizado, on-brand y accionable — igual que Waymark lo hace para video ads, pero para el pipeline completo de marketing y ventas.
